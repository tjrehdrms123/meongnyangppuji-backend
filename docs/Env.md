# Env Sample
```bash
# app
NODE_ENV # 실행 모드(development|production)
PORT # 실행 포트
ADMIN_USER # Swagger DOCS 접속 계정
ADMIN_PASSWORD # Swagger DOCS 접속 계정
DOMAIN_LOCAL # Swagger에 표시되는 Local 도메인
# JWT
SECRET_KEY # JWT Salt SECRET KEY
# DB
DB_USERNAME
DB_PASSWORD
DB_HOST
DB_PORT
DB_NAME
DB_TYPE # DB 종류(현재: mysql)
# SESSION
SESSION_SECRET_KEY # Express Session Key
```
